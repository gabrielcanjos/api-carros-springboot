# 🚗 API REST de Carros - Spring Boot

Projeto desenvolvido para praticar conceitos de API REST com Java e Spring Boot.

## 🔧 Tecnologias
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Lombok
- ModelMapper

## 📌 Funcionalidades
- Listar carros
- Buscar por ID
- Buscar por tipo
- Cadastrar carro
- Atualizar carro
- Deletar carro

## 📡 Endpoints
GET /api/v1/carros  
GET /api/v1/carros/{id}  
GET /api/v1/carros/tipo/{tipo}  
POST /api/v1/carros  
PUT /api/v1/carros/{id}  
DELETE /api/v1/carros/{id}

## 🎯 Objetivo
Praticar construção de APIs REST seguindo boas práticas como uso de DTO e ResponseEntity.
